Development and testing of new illumination correction in DarSIA.

## Getting started
```bash
python gui.py
```

## Learning material

1. Read [introduction.md](docs/introduction.md) for workflow overview
2. Explore [image_selection.md](docs/image_selection.md) to understand configuration options
3. Learn [analysis.md](docs/analysis.md) for available analysis tools
4. Check [finger_analysis.md](docs/finger_analysis.md) for fingering instability studies

## Config files

1. Check out [common.toml](config/common.toml) for common settings
2. Check out [run_exp_3.toml](config/run_exp_3.toml) for run-specific configurations
3. Check out [data.toml](config/data.toml) for selecting data and ROIs
4. Check out [analysis.toml](config/analysis.toml) for analysis configuration
5. Check out [utils.toml](config/utils.toml) for util functionalities

## Data files
1. Check out [depth_measurements.csv](data/depth_measurements.csv) containing pointwise depth measurements
2. Check out [segmented_DSC00160.png](data/segmented_DSC00160.png) containing the manually segmented baseline image
3. Check out [facies.xlsx](data/facies.xlsx) for containing facies information
4. Check out [protocol_exp_3.xlsx](data/protocol_exp_3.xlsx) for the protocols for Experiment 3.
